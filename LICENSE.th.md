## 1. คำจำกัดความ

- **"ซอฟต์แวร์"** — โปรแกรม IVS ทั้งหมด รวมถึง backend, frontend, scripts, การตั้งค่า เอกสาร และการอัปเดต
- **"ผู้ให้สิทธิ์"** — IVS Project เจ้าของลิขสิทธิ์
- **"ผู้รับอนุญาต"** — บุคคลธรรมดาหรือนิติบุคคลเดียวที่ติดตั้ง/ใช้งานซอฟต์แวร์
- **"Free Edition"** — รุ่นแจกฟรี สำหรับใช้งานส่วนตัว/ภายในองค์กรแบบไม่แสวงหากำไร บนเครื่องเดียวที่ระบุด้วย machine fingerprint

## 2. สิ่งที่ทำได้ (Grant of License)

ผู้รับอนุญาตสามารถ:

- ติดตั้งและใช้งาน **1 copy** บน **1 เครื่อง** เพื่อการใช้งานส่วนตัวหรือภายในองค์กรแบบไม่แสวงหากำไร
- Deploy แอปพลิเคชันของผู้ใช้บนซอฟต์แวร์ภายใน private network
- แก้ไข source code **เฉพาะเพื่อ fix bug หรือปรับใช้ในสภาพแวดล้อมของตนเอง** โดยต้อง **คง copyright notice และ license header ไว้ครบถ้วน**

## 4. ทรัพย์สินทางปัญญา

ซอฟต์แวร์ทั้งหมด รวมถึง source code, architecture, UI, รูปแบบ audit log, ระบบ serial number, ระบบ machine fingerprint, รูปแบบ export bundle (v2 manifest) และเอกสารประกอบ เป็นกรรมสิทธิ์ของผู้ให้สิทธิ์แต่เพียงผู้เดียว และได้รับความคุ้มครองภายใต้กฎหมายลิขสิทธิ์ ความลับทางการค้า และสนธิสัญญาระหว่างประเทศ

**ไม่มีการโอนกรรมสิทธิ์** ใดๆ ไปยังผู้รับอนุญาตภายใต้สัญญานี้

แอปพลิเคชันของผู้ใช้ที่ deploy ลงบนซอฟต์แวร์ยังคงเป็นของผู้สร้างแอปนั้น — ข้อ 3.3 ไม่ครอบคลุมแอปของผู้ใช้ บังคับใช้เฉพาะตัว IVS เท่านั้น

## 5. Machine Fingerprint และ Anti-Tamper

เมื่อรันครั้งแรก ซอฟต์แวร์จะสร้าง machine fingerprint และ serial number เฉพาะ เก็บไว้ที่ `~/.ivs/license.bound` — ใช้เพื่อ:

1. ระบุ installation นี้สำหรับ support และ upgrade
2. ตรวจจับการ clone installation ไปยัง hardware อื่นโดยไม่ได้รับอนุญาต
3. รองรับการอัปเกรดเป็น Enterprise

**การแก้ไข ปลอมแปลง ลบ หรือ duplicate** machine fingerprint/serial หรือการรันโดยมีสถานะ fingerprint mismatch ติดต่อกัน **เกิน 30 วัน** ถือเป็นการละเมิดสัญญาขั้นรุนแรง และทำให้สัญญานี้สิ้นสุดโดยอัตโนมัติ
